# Silia-Glasses
Bone conduction smart glasses 3d model files.

They are made from taking apart a really cheap pair of smart glasses stealing the bluetooth module and thin batterys. then using bone conduction speakers placed by the temple to vibrate the music in the same way that a hearing aid works.

This repository is for the 3d print files if anyone ever decides to make anything similar and its cool, the dimension are customized to work with bd02 bone conduction model if those are the ones your urself

basic print instuction:
Download or fork the files, Load up model in Cad platform of choice to view.
Then download onto ur computer again if made any changes.
Go to your slicer load your printer place on board the slice 
print!
